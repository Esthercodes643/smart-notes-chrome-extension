# 📝 Smart Notes — Chrome Extension

A lightweight Chrome extension for capturing and managing notes without leaving your browser.
Built with a custom notebook-scribble aesthetic — ruled lines, Kalam font, dark theme.

## Features

- ✏️ Write and save notes instantly from any tab
- 📌 Pin important notes to keep them at the top
- 🔍 Search and filter notes in real time
- 🗑️ Delete notes individually
- 💾 Notes sync across Chrome sessions via `chrome.storage.sync`
- 🎨 Custom notebook UI — Kalam font, ruled lines, dark theme

## Tech Stack

| Layer | Tech |
|-------|------|
| Logic | Vanilla JavaScript |
| UI | HTML, CSS |
| Storage | Chrome Storage API (Manifest V3) |
| Font | Kalam (Google Fonts) |

## Installation

```bash
git clone https://github.com/Esthercodes643/smart-notes-chrome-extension.git
```

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked** → select the project folder
4. Click the Smart Notes icon in your toolbar and start writing
